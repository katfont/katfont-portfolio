User Flows
Course: Information Architecture, M.S. Information Design and Strategy, Northwestern University
The objective of this project was to develop a sitemap, navigation, and wireframes that illustrated a path toward fulfilling a user goal. This was a cumulative project that started with content collection, markup, and developing a site map. Later elements to this project included identifying user goals, developing user flows, and wireframes.
